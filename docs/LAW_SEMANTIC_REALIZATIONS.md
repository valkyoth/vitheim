# Law Semantic Realization Registry

Status: normative planning registry introduced by `0.18.3`

Canonical semantic text is evidence and review context, never an interpreted
security program. Runtime code exhaustively matches a supported
`VIT-LSEM-NNN-gNN-vN` to compiled Rust transitions, typed outcomes, recovery
logic, and concrete positive/negative/model/fault tests. Unknown, unsupported, or
manifest/realization-mismatched semantic IDs fail startup and restore before
authority is admitted.

Rows are machine-readable and append-only. A path is the planned owning source
location; it must exist and its named test contracts must resolve when the
realization gate leaves planned status. That gate is the later of the
generation's effective milestone and the `0.18.3` realization-registry
bootstrap, so historical generations are backfilled atomically at `0.18.3`.
The closed enum and dispatch table live in
`crates/vitheim-law/src/semantic.rs`; concrete test IDs live in
`crates/vitheim-law/tests/semantic_contracts.rs`. Unchanged semantic text still
receives a generation-specific realization record so manifests, admission
ancestry, binaries, and test evidence cannot disagree about support.

For VIT-LAW-008@g02, `...CapacityDrainFenceInstalled` and
`...CapacityDrainFenceConsumed` are sealed events emitted only inside the
PendingDrain and activation/rejection transactions. They are not commands or
public operations. No `Install...DrainFence` or `Clear...DrainFence` semantic
ID may be introduced; unknown legacy IDs deny.

At `0.18.3`, implement a closed Rust `LawSemanticId` enum/decoder and a
`LawSemanticRealization` dispatch table; there is no dynamic prose interpreter,
string-to-command reflection, or plugin extension point. Each later generation
adds its enum variant, transition implementation, recovery implementation, and
P/N/M/F realization before activation. `0.22.0` adapter conformance enumerates the
full predecessor closure and invokes every listed test contract. Phase O and
`1.0.0` require zero planned/unresolved realizations for shipped generations.
